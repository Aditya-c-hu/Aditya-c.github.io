Algorithms Used in the Code

Dijkstra's Algorithm

Dijkstra's algorithm is used to find the shortest path from a starting vertex to all other vertices in a graph with non-negative weights.

Efficiency: O(V + E log V) where V is the number of vertices and E is the number of edges.

Floyd-Warshall Algorithm

The Floyd-Warshall algorithm is used to find shortest paths between all pairs of vertices in a weighted graph.

Efficiency: O(V³) where V is the number of vertices.

Kruskal's Algorithm

Kruskal's algorithm is used to find the minimum spanning tree (MST) for a connected graph.

Efficiency: O(E log E) where E is the number of edges.

Quick Sort

Quick Sort is an efficient sorting algorithm that employs a divide-and-conquer approach.

Average Efficiency: O(n log n) where n is the number of elements.

Worst-case Efficiency: O(n²) when the pivot selection is poor.

Knuth-Morris-Pratt (KMP) Algorithm

The KMP algorithm is used for substring search within a main string.

Efficiency: O(n + m) where n is the length of the main string and m is the length of the pattern.

Rabin-Karp Algorithm

The Rabin-Karp algorithm is used for searching multiple patterns within a main string using hashing.

Efficiency: O(n + m) on average and O(nm) in the worst case where n is the length of the main string and m is the length of the pattern.

Heap Sort

Heap Sort is a comparison-based sorting technique based on a binary heap data structure.

Efficiency: O(n log n) where n is the number of elements.

Min Heap

A Min Heap is a binary tree where the parent node is less than or equal to its children, used to efficiently retrieve the minimum element.

Efficiency: O(1) for getting the minimum element, O(log n) for insertions and deletions where n is the number of elements.
